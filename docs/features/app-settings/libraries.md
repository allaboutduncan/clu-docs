# Libraries

![Libraries](../../assets/settings/libraries.png){: .center-image}

Libraries are the root of your collection. 

You can add, edit, pause and remove libraries from this page.

## Add Library

![Add Library](../../assets/settings/add-library.png){: .center-image}

Click the **Add Library** button to add a new library. You will be prompted to enter the library name and the path to the library. The path will be the mapping to the drive you specified during installation.

## Edit Library

![Edit Library](../../assets/settings/edit-library.png){: .center-image}

Click the **Edit** icon <i class="bi bi-pencil text-info"></i> to edit a library. You will be prompted to enter the library name and the path to the library. The path will be the mapping to the drive you specified during installation.

## Pause Library

![Pause Library](../../assets/settings/pause-library.png){: .center-image}

Click the **Pause** icon <i class="bi bi-pause text-warning"></i> to pause a library. This will prevent the library from being scanned and showing on the collection page.

Click the **Resume** icon <i class="bi bi-play text-success"></i> to resume a library. This will allow the library to be scanned and show on the collection page.

## Remove Library

Click the **Remove** icon <i class="bi bi-trash text-danger"></i> to remove a library.

You will be prompted to confirm the removal of the library. This will remove the library from the app. **Files are not deleted from the filesystem.**
