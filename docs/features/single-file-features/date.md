# Add Read Date

![Add Read Date](../../assets/single/date01.png){: .center-image }

Accessible in the only in the [Collection view](../collection/index.md), via the <i class="bi fs-2 bi-three-dots-vertical text-dark"></i> dropdown menu. 

Use this tool to add or update the date the comic was read.

As you read comics, the date data will be used to populate the reading activity and heatmap on the [Library Insights](../insights/insights.md) page.
