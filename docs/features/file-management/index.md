# File Management

![Built in File Manager](../../assets/file/file-manager.png)

Manage your entire library remotely and Move, Rename and Delete directories and files all from one easy drag-and-drop interface.

!!! info "Realtime File Manager"
    The **File Manager** feature does not use the database to list or store file information. Response time is based on the size of the directory and the number of files in it.
