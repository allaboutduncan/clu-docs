# ComicVine Metadata

If you have entered a [Commicvine API Key](/features/gcd-settings/) in settings, you'll see an additional icon in the File Manager to search for metadata from ComicVine. Clicking the <i class="bi bi-book fs-2 text-success"></i> icon will beging searching for metadata from ComicVine.

### Get Metadata for an Issue

Clicking the <i class="bi bi-book fs-2 text-success"></i> icon will open a modal where you can search for metadata from ComicVine.

### Metadata on File Move

When you move a file to a new location, you can use the <i class="bi bi-book fs-2 text-success"></i> icon to search for metadata from ComicVine.

### Metadata on File Rename

When you rename a file, you can use the <i class="bi bi-book fs-2 text-success"></i> icon to search for metadata from ComicVine.
